Introduction

Bienvenue dans ce guide complet pour apprendre à créer un site web en HTML et CSS ! Ce guide couvre les bases et quelques techniques avancées pour vous aider à créer des pages web modernes et interactives.

Ajouter une Image

Pour ajouter une image, on utilise la balise <img>. Voici un exemple :

<img src="image.jpg" alt="Une belle image" width="600" height="400">

Ajouter un Formulaire

Les formulaires permettent de recueillir des informations auprès des utilisateurs. Voici un formulaire simple :

<form action="traiter.php" method="POST">

<label for="nom">Nom:</label>

<input type="text" id="nom" name="nom">

<label for="email">Email:</label>

<input type="email" id="email" name="email">

<button type="submit">Envoyer</button>

</form>

Ce formulaire contient des champs pour le nom et l'email, ainsi qu'un bouton pour soumettre les données.

Créer une Table

Les tableaux sont utilisés pour organiser des informations sous forme de grille. Voici un exemple :

<table>

<tr>

<th>Nom</th>

<th>Âge</th>

</tr>

<tr>

<td>Jean</td>

<td>28</td>

</tr>

</table>

Le tableau contient deux colonnes : "Nom" et "Âge", avec une ligne de données.

Ajouter un Lien

Les liens permettent de naviguer entre les pages ou vers des ressources externes. Voici un exemple de lien :

<a href="https://www.example.com">Visitez notre site</a>

Visitez notre site

Ajouter une Vidéo

Les vidéos peuvent être intégrées à votre page avec la balise <video>. Voici un exemple :

<video width="600" controls>

<source src="video.mp4" type="video/mp4">

</video>

Voici une vidéo intégrée :

Footer

Le pied de page (footer) est souvent utilisé pour des informations légales ou des liens supplémentaires. Voici un exemple simple :

<footer>

<p>© 2025 Mon Site Web - Tous droits réservés</p>

</footer>

Le pied de page (footer) contient des informations légales.

Ajouter des Boutons Interactifs

Les boutons permettent aux utilisateurs d'interagir avec votre site. Voici un exemple de bouton :

<button class="interactive-button">Cliquez-moi</button>

Ajouter un Menu Déroulant

Un menu déroulant peut être utile pour afficher des options supplémentaires sans encombrer la page. Voici un exemple :

<div class="menu-dropdown">

<button>Menu</button>

<div class="dropdown-content">

<a href="#">Option 1</a>

<a href="#">Option 2</a>

</div>

</div>

JavaScript Interactif

JavaScript permet de rendre votre site interactif. Voici un exemple simple pour changer le texte d'un bouton lorsqu'il est cliqué :

<button onclick="changerTexte()">Cliquez pour changer le texte</button>

<script>

function changerTexte() {

  document.querySelector('button').innerText = 'Vous avez cliqué !';

</script>
